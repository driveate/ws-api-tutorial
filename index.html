Overview

You are looking at the API Documentation and still don't know how it may be useful for your needs?

Let's learn it by comparing API data with information on the site.

Each part of this tutorial arranged so that you need three simple steps to understand API methods:

1. Look at the screenshot from wheel-size.com. Each screenshot contains one self-sufficient information block.
2. Learn API method calls and returned data that fit this screenshot.
3. Compare the data returned by API call and what you see on the image above and read auxiliary comments.

Following visualization sings are useful to understand tutorial information:

tire denotes data from screenshot
name denotes field from JSON
make denotes parameter from API method
/models/ denotes API method

connects screenshot with corresponding JSON

means beginning of the next logical block

denotes a block with some additional information
